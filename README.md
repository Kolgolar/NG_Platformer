## Управление:
### Передвижение:
[W][A][S] / [Стрелочки] (вправо, влево, прыжок).

Персонаж может совершать *двойной прыжок*.
### Атака:
- Прыжки на противников сверху
- Нажатие клавиши [Пробел]

### Прочее:
Уровень всегда можно перезапустить, нажав и удерживая клавишу [R]
